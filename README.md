# 📊 Bank Loan Dashboard

## 📌 Project Overview
The **Bank Loan Dashboard** is an Excel-based data analytics project designed to analyze and monitor loan applications, funded amounts, repayments, and customer information. The dashboard provides meaningful insights into loan performance and helps in understanding trends, risk factors, and customer behavior.

---

## 🎯 Objective
The main objective of this project is to:

- Analyze bank loan data efficiently.
- Monitor loan applications and funding status.
- Track loan repayment performance.
- Identify good and bad loans.
- Provide visual insights for better decision-making.

---

## 📂 Dataset Information
The dataset contains information related to bank loans, including:

- Loan ID
- Customer Details
- Loan Amount
- Funded Amount
- Interest Rate
- Loan Status
- Grade
- Purpose of Loan
- Employment Length
- Home Ownership
- Annual Income
- Issue Date
- State and Region Information
- Term of Loan

---

## 📈 Dashboard Features

### 1. Summary Dashboard
Provides an overall view of key performance indicators (KPIs):

- Total Loan Applications
- Total Funded Amount
- Total Amount Received
- Average Interest Rate
- Average Debt-to-Income Ratio (DTI)
- Good Loan Percentage
- Bad Loan Percentage

### 2. Overview Dashboard
Displays trends and patterns using charts and visualizations:

- Monthly Loan Application Trend
- Loan Applications by State
- Loan Applications by Purpose
- Loan Applications by Term
- Loan Applications by Employment Length
- Loan Applications by Home Ownership
- Loan Status Analysis

### 3. Detailed Dashboard
Provides detailed information about individual loan records for deeper analysis.
